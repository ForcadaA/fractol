# Description

`fractol` visualizes escape‑time fractals with real‑time zooming and keyboard controls.  
It supports:
- **Mandelbrot set** (`m`)
- **Burning Ship** (`d`)  
- **Julia set** (`j`) with custom complex seed

# Requirements

- **MinilibX** (the 42 version)
- **X11** / **XQuartz** (on macOS)
- **make**, **gcc** / **clang**
